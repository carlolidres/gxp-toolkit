## Recommended Usage

Use this file as a reference index only.

Do not download all repositories.

Do not merge these repositories directly into the starter template.

Recommended workflow:

1. Open the relevant reference link.
2. Study the UI/UX, folder structure, component design, and implementation pattern.
3. Copy only small patterns when needed.
4. Rewrite and adapt the code to match this project’s own structure.
5. Keep this project clean, reusable, and independent.
